# API REST - Sistema de Recursos Humanos

API desarrollada con Spring Boot para la gestión de empleados.

---

## Funcionalidades

- CRUD de empleados
- Gestión de datos laborales
- Persistencia en base de datos

---

## Tecnologías

- Java
- Spring Boot
- Spring Data JPA
- MySQL / H2
- Maven

---

## Endpoints

- GET /api/empleados
- POST /api/empleados
- PUT /api/empleados/{id}
- DELETE /api/empleados/{id}

---

## Cómo ejecutar

1. Clonar el repositorio
2. Ejecutar la clase principal
3. Acceder a:
http://localhost:8080/api
